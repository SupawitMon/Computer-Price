# 💻 Computer Price Prediction

## Problem
Predict computer prices based on specifications.

## Dataset
All Computer Prices dataset from Kaggle.

## Approach
- Pipeline for preprocessing
- RandomForest Regressor
- Hyperparameter tuning (RandomizedSearchCV)
- Cross-validation

## Metrics
- MAE
- RMSE
- R2 Score

## Deployment
Streamlit App

## Disclaimer
Predictions are estimates only.
